# Employee Management System

This project is an Employee Management System designed to manage employee records efficiently. It supports basic operations such as adding, viewing, updating, and deleting employee details.

## Features
- Employee record management
- Structured backend logic
- Simple frontend interface
- Data persistence
- Clean and modular project structure

## Tech Stack
- Backend: Python
- Frontend: HTML, CSS, JavaScript
- Database: File-based / local storage
